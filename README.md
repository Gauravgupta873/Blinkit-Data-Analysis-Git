# 🛒 Blinkit Sales Data Analysis Dashboard

## 📌 Project Overview
The Blinkit Sales Data Analysis Dashboard** is a Power BI project that analyzes sales performance, outlet characteristics, and product categories for Blinkit grocery stores.

The dashboard provides insights into total sales, average sales, product performance, outlet sizes, outlet locations, and customer ratings using interactive visualizations.

This project demonstrates how data analytics and visualization can help businesses understand sales trends and make better decisions.

---

## 🚀 Features

### 📊 Sales Analysis
- Total sales overview
- Average sales per item
- Sales trends across outlet establishment years

### 📦 Product Insights
- Number of items available
- Item type performance comparison
- Item fat content analysis

### 🏬 Outlet Analysis
- Sales by outlet size (Small, Medium, High)
- Sales by outlet location (Tier 1, Tier 2, Tier 3)
- Outlet type performance comparison

### ⭐ Customer Insights
- Average product rating
- Item visibility metrics
- Performance comparison between outlet types
